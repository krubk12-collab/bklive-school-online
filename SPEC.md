# โรงเรียนออนไลน์ (BK Live Map) — สเปก

**คอนเซ็ปต์:** แผนที่โรงเรียนมองจากบน ครู 50+ คนเป็นชิปสีตามกลุ่มสาระ ขยับไปตามห้องที่สอนจริงในแต่ละคาบ เลื่อนเวลาดูได้ทั้งวัน / กด "ตอนนี้" sync นาฬิกาจริง เป็น visualization ตารางสอน ไม่ใช่เกมแข่งขัน

## Stack
- ไฟล์ HTML เดียว vanilla JS + CSS, วางบน GitHub Pages ได้ ไม่มี framework/build/game engine

## ข้อมูล 3 ก้อน (JSON)
- `teachers` — ชื่อ, ชื่อย่อ, กลุ่มสาระ (→ สี)
- `timetable` — [ครู][วัน][คาบ] = { วิชา, ห้อง }
- `rooms` — ห้อง + พิกัด x,y,w,h + ประเภท (เฟส 2 มีโหมดจัดผังลากได้)
- เฟส 1: ข้อมูลทั้งหมด mock ในไฟล์ · เฟส 2: export static JSON จาก GAS ครั้งเดียว

## ส่วนประกอบ (หน้าที่เดียวต่อตัว)
| ตัว | ทำอะไร |
|---|---|
| timeControl | แถบเลื่อนเวลา + ปุ่ม "ตอนนี้" + เลือกวัน → คาบปัจจุบัน |
| placement | คาบ+ครู → ห้อง → พิกัด (ครูห้องเดียวกันกระจายไม่ทับ) |
| mapRender | วาดกล่องห้อง |
| teacherLayer | ชิปครูขยับด้วย CSS transition (ลื่นไถลเหมือนเดิน) |
| roomEditor | (เฟส 2) ลากจัดห้อง เซฟ JSON |
| panel | คลิกครู/ห้อง → รายละเอียด |

## กันพลาด
- ครูไม่มีคาบ → โซนห้องพักครู/โรงอาหาร (ไม่หาย)
- ห้องในตารางไม่มีในผัง → โซน "ยังไม่จัดผัง" + แจ้งเตือน
- โหลดข้อมูลไม่ได้ → mock + แจ้ง

## กิจกรรมพิเศษ
พละ→สนาม · ศิลปะ→ห้องศิลป์/ดนตรี · คอม→ห้องคอม · วิทย์→ห้องวิทย์ · ที่เหลือ→ห้องประจำชั้น

## เฟส
1. **เฟส 1 (นี่)** — mock + แผนที่ + ครูขยับ + timeline
2. เฟส 2 — ข้อมูลจริง + โหมดจัดผัง
3. เฟส 3 — ลูกเล่น (ไฮไลต์คาบว่าง, กรองกลุ่มสาระ)

## ตัดทิ้ง (YAGNI)
คะแนน/เลเวล · ล็อกอิน · save · เสียง
